Features of Time travel and cost estimator 
1.Enter starting location
2.Enter destination location
3.Input total travel distance (km)
4.Input vehicle mileage (km/l)
5.Input fuel price per liter
6.Input average speed (km/h)
7.Calculate estimated travel time
8.Calculate total fuel needed
9.Calculate total travel cost
10.Display summary in neat formatted output
11.Allow multiple trips in one run
12.Use methods/functions for calculations
13.Validate inputs (no negative values)
14.Show travel time in hours and minutes
15.Offer option to compare two travel routes
16.Calculate cost difference between routes
17.Option to clear screen and restart
18.Use class and object concepts
19.Store trip details in a list (optional)
20.Print a final report of all trips entered
